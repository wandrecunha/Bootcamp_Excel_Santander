PERGUNTAS DE NEGÓCIO:

O desafio consiste em construir uma planilha que ajude o usuário a realizar essas simulações, auxiliando-o a tomar decisões mais informadas sobre seus investimentos. 

1. valor total investido;
2. o patrimônio acumulado;
3. dividendos mensais;
4. Visão clara quanto aos investimento futuro.

Objetivo é criar um simulador simples de investimento.

1. Quanto investir por mês;
2. Por quantos anos tem que investir;
3. Taxa de rendimento mensal;
4. Apresentar uma visão no tempo;

Fórmulas utilizadas:

1. Qual patrimônio total (quanto tem acumulado): vf(Taxa de Rendimento mensal); (Por quantos anos) * 12; (Aporte) * -1) 
2. Dividendos mensais: Total de patrimonio * 1% ao mês
3. Criação de Tabela Auxiliar para consolidar a porcentagem de cada perfil de investidor (Aplicação da fórmula PROCV)
4. Utilização da Validação de dados para padronizar a informação (perfil do investidor) e possibilitar a construção de chave.
